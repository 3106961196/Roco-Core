# RocoCore - 洛克王国世界辅助核心

> **XRK-AGT框架的洛克王国世界游戏辅助模块**
> 
> 当前版本：Phase 1 (MVP) - 远行商人功能

## 📋 功能特性

### ✅ Phase 1 已实现功能

- **🔍 远行商人查询** - `#远行商人` / `#商人`
  - 实时获取当前轮次商品信息
  - 精美卡片式UI展示（Tailwind CSS）
  - 包含今日历史时段数据
  - 实时倒计时显示
  - 浏览器截图输出

- **⏰ 智能检测系统**
  - 延迟1分钟开始检测（避免刷新延迟）
  - 到点后每分钟自动检测一次
  - 获取成功后停止，等待下次刷新
  - 智能缓存机制（30分钟有效期）

- **📊 状态监控** - `#商人状态` / `#商人检测`
  - 显示当前轮次和时段
  - 缓存状态查看
  - 系统运行状态

- **🔄 强制刷新** - `#刷新商人` （管理员权限）
  - 清除缓存并重新获取
  - 强制更新最新数据

## ⚙️ 使用说明

### 前置要求

- Node.js ≥ 24.13.0
- XRK-AGT 框架已正确配置
- Playwright 浏览器已安装（Chromium）

### 安装步骤

1. **确认目录位置**
   ```
   插件位于: core/RocoCore/plugin/roco-merchant.js
   ```

2. **配置文件**（可选）
   编辑 `config/roco.yaml` 自定义设置：
   ```yaml
   roco:
     merchant:
       cache:
         ttl: 1800  # 缓存时间（秒）
       ui:
         width: 820  # 渲染宽度
         imageQuality: 90  # 图片质量
   ```

3. **启动XRK-AGT**
   ```bash
   node start.js
   ```

### 使用指令

| 指令 | 说明 | 权限 |
|------|------|------|
| `#远行商人` 或 `#商人` | 查询当前商品 | 所有人 |
| `#商人状态` 或 `#商人检测` | 查看系统状态 | 所有人 |
| `#刷新商人` | 强制刷新数据 | 管理员 |

## 🔧 核心技术特性

### 1. 智能检测策略

```
时间轴示例（以12:00轮次为例）：
12:00 ── 商人官方刷新
12:01 ── 🟢 开始智能检测（延迟1分钟）
12:02 ── 第1次检测 → 失败？继续
12:03 ── 第2次检测 → 成功！✅ 停止检测
...
16:00 ── 下一轮次，重复上述流程
```

**优势**：
- ✅ 避免官方刷新延迟导致的数据空白期
- ✅ 智能重试，最多尝试30次（30分钟）
- ✅ 成功即停，节省资源
- ✅ 自动调度下一轮检测

### 2. 数据流架构

```
用户指令 → 插件接收 → 检查缓存
                    ↓
              [缓存有效？]──是──→ 返回缓存数据
                    ↓否
              启动爬取器
                    ↓
          Playwright访问网页
                    ↓
          提取并解析数据
                    ↓
          写入缓存 + 渲染UI
                    ↓
          返回图片给用户
```

### 3. UI渲染流程

```
数据对象 → 注入HTML模板 → Playwright渲染 → 截图输出
                ↓
    Tailwind CSS样式化
                ↓
    实时倒计时计算
                ↓
    图标本地优先加载
                ↓
    输出JPEG图片（820px宽）
```

## 🐛 常见问题

### Q: 插件无法加载？
**A:** 检查以下项：
1. 文件路径是否正确：`core/RocoCore/plugin/roco-merchant.js`
2. Node.js版本是否符合要求（≥24.13）
3. 是否有语法错误（查看控制台日志）

### Q: 爬取失败？
**A:** 可能原因：
1. 网络连接问题
2. 目标网站反爬虫机制触发
3. Playwright浏览器未正确安装

解决方案：
```bash
# 重试或检查网络
# 刷新商人 #管理员指令
```

### Q: 图片无法显示？
**A:** 检查：
1. `www/merchant/assets/` 目录下资源文件是否存在
2. Playwright截图功能是否正常
3. 查看日志中的错误信息

### Q: 如何调整检测时间？
**A:** 编辑 `plugin/shared/time-utils.js`：
```javascript
const DETECTION_DELAY_MINUTES = 1  // 修改此值（单位：分钟）
```
